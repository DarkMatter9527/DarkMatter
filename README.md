# 基于视觉引导与强化学习的双模态光谱矿物智能扫描策略研究
本项目面向矿物光片智能识别场景，融合计算机视觉分割、短波红外（SWIR）光谱分类与强化学习决策，实现「快速粗筛 + 精确复核」的双模态智能扫描策略，在保证识别精度的同时降低整体检测耗时。

## 代码模块说明
### 1. `sam3Extract.py` — 矿物图像分割与特征提取
基于 SAM3 实例分割模型，对显微镜下的矿物光片大图进行自动分割：
- 生成网格提示点驱动 SAM3 检测，通过 NMS 去重得到独立矿物颗粒掩码
- 提取每个颗粒的几何特征（面积占比、质心坐标、长宽比、紧凑度）与颜色特征（RGB/HSV 均值）
- 输出分割可视化结果图与颗粒特征 CSV 文件

### 2. `dataTabeling.py` — 真值标注对齐与数据集构建
将像素级矿物真值标注图与分割结果做坐标映射，为每个颗粒打上真实类别标签：
- 读取 `.npy` 格式的真值标签图，按颗粒质心坐标匹配真实矿物类别
- 剔除背景样本，生成带真实标签的训练数据集 CSV
- 附带生成「原图-分割图-真值标注」三图对比、面积占比饼图等可视化分析图表

### 3. `SWIRClassifier.py` — SWIR 高光谱分类器
基于光谱角匹配（SAM）算法实现 SWIR 高光谱矿物分类：
- 构建五类矿物的标准参考光谱库，将颗粒坐标映射到 SWIR 光谱立方体
- 通过余弦相似度计算光谱角，取最小夹角类别作为识别结果
- 将光谱角归一化为 0~1 置信度，输出带分类结果与置信度的强化学习输入数据集
- 附带生成混淆矩阵等分类效果评估图表

### 4. `RLTrain.py` — GRPO 强化学习训练与评估
构建矿物扫描决策环境，使用 GRPO（分组相对策略优化）算法训练智能体：
- 定义双动作空间：SWIR 快速检测（低成本、有误差）、Raman 精确检测（高成本、无误差）
- 奖励函数融合精度收益、时间惩罚与连续动作奖励，平衡识别速度与准确率
- 实现残差结构策略网络，支持分组策略更新、熵正则化与学习率衰减
- 内置完整评估体系：输出准确率/耗时对比、混淆矩阵、分矿物类别指标、置信度-动作关联分析等多维度结果图表

## 整体运行流程
1. **图像分割**：运行 `sam3Extract.py`，输入矿物光片原图，输出颗粒特征 CSV 与分割可视化
2. **真值映射**：运行 `dataTabeling.py`，匹配真值标注，生成带标签的基础数据集
3. **光谱分类**：运行 `SWIRClassifier.py`，结合 SWIR 高光谱数据，生成带分类置信度的强化学习训练集
4. **策略训练**：运行 `RLTrain.py`，训练 GRPO 智能体，输出最优策略模型权重与全套评估分析图表

## 核心输出
- 矿物颗粒特征数据集、带标签与光谱置信度的强化学习训练集
- 训练好的双模态决策策略模型权重
- 分割效果、分类精度、训练曲线、策略对比等全套论文级可视化图表
